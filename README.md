# 🎡 幸运转盘 - 抽奖奖励工具

一个简单易用的 Web 抽奖转盘应用，专为家长给孩子设置任务奖励而设计。

## ✨ 功能特点

- **自定义选项**：支持 3-5 个奖励选项，内容完全自定义
- **概率设置**：每个选项可设置抽中概率（百分比，合计 100%）
- **三种主题**：简洁、卡通、中国民族风三种视觉风格
- **抽奖限制**：每次 3 次抽奖机会，用完后可重置
- **动画效果**：流畅的转盘动画 + 烟花庆祝特效
- **本地保存**：可导出为单个 HTML 文件，方便分享和离线使用
- **可编辑**：支持随时返回修改配置

## 🚀 快速开始

### 方法一：直接使用

1. 双击打开 `index.html` 文件
2. 按照屏幕提示完成配置：
   - 第一步：输入奖励选项和概率
   - 第二步：设置标题和选择主题
   - 第三步：开始抽奖！

### 方法二：本地服务器

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 然后在浏览器打开
# http://localhost:8000
```

## 📖 使用说明

### 第一步：设置奖励选项

1. 输入 3-5 个奖励选项（如：游戏时间、零用钱、看电影等）
2. 设置每个选项的抽中概率（百分比）
3. 确保所有选项概率合计为 100%
4. 点击"下一步"

### 第二步：设置标题和主题

1. 输入转盘标题（如：宝贝奖励大抽奖）
2. 选择喜欢的主题风格：
   - **简洁**：清爽干净的蓝绿色调
   - **卡通**：活泼可爱的彩虹配色
   - **中国民族风**：传统喜庆的红金色调
3. 点击"生成转盘"

### 第三步：开始抽奖

1. 点击"开始抽奖"按钮
2. 转盘会旋转 3 秒钟
3. 停止后弹出中奖结果，伴随烟花特效
4. 每次有 3 次抽奖机会，用完后可重置

### 编辑和保存

- **编辑**：点击"编辑"按钮返回配置界面，可修改所有设置
- **保存**：点击"保存到本地"按钮，将转盘导出为 HTML 文件

## 💡 使用技巧

1. **概率设置建议**：
   - 高价值奖励（如大额零用钱）设置较低概率
   - 普通奖励（如游戏时间）设置较高概率
   - 保证所有概率合计正好 100%

2. **主题选择建议**：
   - 日常使用推荐"简洁"主题
   - 生日或节日推荐"卡通"或"中国民族风"主题

3. **保存和分享**：
   - 保存的 HTML 文件可以直接发给别人使用
   - 文件包含所有配置，打开即可使用

## 🔧 技术说明

- **纯前端实现**：HTML + CSS + JavaScript，无需后端服务器
- **响应式设计**：支持电脑和手机浏览器
- **本地存储**：使用 localStorage 保存配置数据
- **Canvas 动画**：使用 Canvas API 实现烟花特效
